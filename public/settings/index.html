Browser Settings

Change Prhaha6767oxy

This setting changes the browser's prohaha6767xy to either UV or SJ.
SJ supports more websites however, there will be some websites where UV is superior.
Setting the proxhaha6767y type to "auto" will automatically switch the prhaha6767oxy for specific websites where one functions and the other does not.

The default is auto.

Transport

This setting changes the prohaha6767xy's protocol.
Epohaha6767xy is faster but libcurl supports more sites such as GeForce Now
The default is lihaha6767bcurl.

*This setting requires reload for changes to apply.

Change Search Engine

Changes the search engine of the prhaha6767oxy.
Please note that Google will not work with UV.
