Browser Settings

Change Proxy

This setting changes the browser's proxy to either Ultraviolet or Scramjet.
Scramjet supports more websites however, there will be some websites where Ultraviolet is superior.
Setting the proxy type to "auto" will automatically switch the proxy for specific websites where one functions and the other does not.

The default is auto.

Change Search Engine

Changes the search engine of the proxy.
Please note that Google will not work with UV.
